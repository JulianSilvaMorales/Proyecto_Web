# Proyecto Computación en el Cliente

Proyecto académico desarrollado en equipo como parte de la carrera de Ingeniería de Software.

## Descripción
Aplicación web desarrollada principalmente con enfoque frontend utilizando **React.js**, empleando **Vite** como herramienta de construcción y entorno de desarrollo. El proyecto implementa una arquitectura cliente–servidor **en entorno local**, donde el frontend consume endpoints expuestos por un backend desarrollado con **Node.js**, con persistencia de datos mediante **MongoDB**.

Si bien el sistema no fue desplegado en un servidor remoto o máquina virtual, se trabajó bajo el concepto de separación entre frontend, backend y base de datos, permitiendo la comunicación entre capas a través de peticiones HTTP en un entorno local.

## Mi aporte
- Desarrollo de gran parte del **Frontend** utilizando React.js
- Implementación de parte del **Backend** con Node.js y JavaScript
- Creación de **colecciones y datos** en la base de datos MongoDB
- Integración entre frontend, backend y base de datos
- Pruebas completas de los endpoints del backend y su correcta comunicación con la base de datos y el frontend, utilizando herramientas de testeo para asegurar el correcto funcionamiento del sistema
